# Sajid Khan Portfolio

A modern, responsive portfolio website built with React.

## Features

- **Navigation Bar**: Logo with profile picture, navigation links (Work, About Me), social media links, and Get in Touch button
- **Project Grid**: Display of 12 project cards with pagination
- **Pagination**: Navigate through projects (12 cards per page)
- **Resume Page**: Contact information, social media links, skills, experience, and education
- **Dark Theme**: Modern dark theme matching the design

## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)
